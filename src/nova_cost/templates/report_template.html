Understanding AWS Cost Explorer Charges
Important: The AWS Cost Explorer charges on your bill are for API calls to the Cost Explorer service, not for using the AWS Console interface.
Each Cost Explorer API request costs approximately $0.01.
CloudTrail Event History View Cost Explorer Events
What's causing these charges?

Something in your environment is programmatically querying the Cost Explorer API. This could be:

  • Custom Scripts or Applications: Code that calls functions like get_cost_and_usage, get_reservation_utilization, etc.
  • Python Boto3 Applications: Using Boto3's client('ce') functions. Note that using Boto3 with other AWS services (like S3, EC2, Lambda) doesn't cause Cost Explorer charges - only direct queries to the Cost Explorer API itself.
  • Third-Party Cost Management Tools: External tools that integrate with your AWS account
  • AWS Solutions or Internal Tools: Dashboards configured to pull cost data via the API
How to investigate:
  1. Set up AWS CloudTrail if not already enabled:
    • Go to the AWS CloudTrail service
    • If you don't have CloudTrail enabled, you'll see a "Create a trail" button - click it
    • Name your trail (e.g., "ApiUsageMonitoring")
    • For "Storage location", create a new S3 bucket or use an existing one
    • Enable "Management events" to capture API calls
    • Click "Next" and then "Create trail"
  2. View CloudTrail events to identify Cost Explorer API calls:
    • Once CloudTrail is set up, go to the CloudTrail console
    • In the left navigation menu, look for and click on "Event history" (not on the main dashboard page)
    • Where to find Event history:
      Look for the navigation menu on the left side of the CloudTrail console with options like:
      • Dashboard
      • Event history ← Click this
      • Trails
      • Lake (if configured)
      • Insights events (if configured)
    • Once in Event history, set a filter for "Event source" = ce.amazonaws.com
    • Look for API calls like GetCostAndUsage, GetReservationUtilization, etc.
    • Click on an event to examine its details
    • In the event details, look at the "userIdentity" section to find which IAM user, role, or service made the API call
Note: CloudTrail has a free tier that includes the management events needed to track Cost Explorer API calls. The trail history is available for 90 days. For longer-term storage, you'll need to configure a trail that saves to an S3 bucket.
How to reduce or eliminate charges:
  1. If it's a custom script/application: Modify or disable the script
  2. If it's a third-party tool: Check the tool's configuration to reduce the frequency of data pulls
  3. If it's an assumed role: Trace back what service or user assumed that role to make the call
Good news: Once you stop the source making the API calls, the Cost Explorer charges on your future bills should cease.
